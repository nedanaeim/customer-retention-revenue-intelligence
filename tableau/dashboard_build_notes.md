# Tableau Dashboard Build Notes

Use the CSV files in `data/processed/` as Tableau data sources.

## Recommended Dashboard Pages

1. Executive overview
   - Total revenue
   - Delivered orders
   - Unique customers
   - Average order value
   - Repeat purchase rate
   - Churn-risk customer share

2. Revenue trends
   - Monthly revenue line chart
   - Monthly orders and customers
   - Average order value trend

3. Retention and customer value
   - Repeat buyer versus one-time buyer split
   - Customer value segment revenue
   - Activity status distribution
   - Cohort retention heatmap

4. Product performance
   - Top categories by merchandise revenue
   - Items sold by category
   - Average item price by category

5. Regional performance
   - Revenue by customer state
   - Orders by customer state
   - Review score by state

## Suggested Tableau Files

When Tableau is available, save the packaged workbook here:

```text
tableau/customer_retention_revenue_dashboard.twbx
```

If publishing to Tableau Public, add the public dashboard link to the main `README.md`.
